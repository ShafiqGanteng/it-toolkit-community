# Changelog

## [3.0.0] - 2026-XX-XX (Community Edition Release)

### Added
- 🎉 Initial public release as Community Edition
- 🖨️ Printer Manager with auto-scan & device name detection
- 🌐 Network tools (Wi-Fi passwords, IP/MAC, speed test)
- 🛡️ System maintenance (cleanup, system info, unhide files)
- 🔧 Common fixes (audio, bluetooth, Windows Update)
- 💾 Driver manager (scan, backup, install)
- 🚀 Quick installer (WinGet integration)
- 🛡️ Security tools (Defender scan, autorun)
- 💼 Backup & restore (profile, bookmarks)
- 👥 User account management
- 🖥️ Remote support (RustDesk)
- ⚙️ Windows tweaks
- 🔧 Utility (password generator)

### Changed
- Removed Office activation features (legal reasons)
- No login required (open access for community)
- Improved menu layout with 3-column display
- Centered banner on terminal

### Security
- Removed sensitive activation scripts
